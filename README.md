# 自主學習時間管理系統

此專案依照 PDF 的功能模組製作：目錄、今日任務、新增事項、生活型態、月曆、甘特圖、時間記錄區、搜索區。

## 使用方式

```bash
npm install
npm start
```

打開：`http://localhost:3000`

## 功能

- 今日任務：依截止時間與重要程度排序，可完成、取消完成、編輯、刪除。
- 新增事項：任務標題、重要程度 1-5、生活型態、開始時間、結束時間、預計時間、子任務、備註。
- 生活型態：可新增顏色與名稱，可刪除；刪除後舊資料仍保留。
- 月曆：每格顯示當日最重要任務，下方顯示學習進度甘特圖。
- 時間記錄：20 分鐘一段，24 小時分成 4 頁，每頁 6 小時，可雙擊編輯。
- 搜索：搜尋任務、時間記錄、生活型態。

## 資料庫

目前前端使用 localStorage，可直接展示與測試。`database/schema.sql` 已附 MySQL 資料表，可之後改接後端資料庫。
